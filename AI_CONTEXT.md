# AI Context

This is a Python-only RAG onboarding assistant.

## Important Files

- `app.py`: Streamlit UI.
- `api.py`: FastAPI endpoints.
- `rag/documents.py`: file loading and chunking.
- `rag/embeddings.py`: BGE embeddings.
- `rag/store.py`: ChromaDB storage and search.
- `rag/chain.py`: Gemini answer generation.

## Rules

- Keep code minimal and beginner-friendly.
- Do not add Java, React, JWT, PostgreSQL, Docker, Ollama, or placeholder CRUD.
- Prefer clear functions over deep architecture.
- Keep docs synced with the simple Python app.
